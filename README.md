# 0️⃣ 机器人基础（Robotics Fundamentals）

本章节帮助学习者建立机器人建模与控制的基础思维，用于理解仿真环境中的所有位姿、关节、模型结构。

---

## 0.1 坐标系与位姿表示

机器人系统中常见的坐标系：

* 世界坐标系（World）
* 机器人基座坐标系（Base）
* 连杆局部坐标系（Link）
* 末端执行器坐标系（End-effector，TCP）
* 相机坐标系（Camera）
* 物体坐标系（Object）

常见的位姿表示方法：

* 平移（x, y, z）
* 旋转（轴角、欧拉角、四元数）
* 位姿矩阵（用于描述一个坐标系相对另一个坐标系的位置与方向）

你将学习：

* 如何描述两个坐标系之间的位置关系
* 如何将点从一个坐标系转换到另一个坐标系
* 如何组合多个变换（例如：Base → Camera → Object）

这些内容在仿真、手眼标定、姿态估计、运动规划中都非常关键。

---

## 0.2 前向运动学（FK）

前向运动学的目的：

> 给定机器人各关节的角度，计算末端执行器（手爪）在空间中的位置与方向。

你将学习：

* 关节是如何带动连杆运动的
* 多个连杆如何逐级连接
* 如何使用 FK 得到末端坐标

仿真环境（如 Isaac Lab）会自动计算 FK，但理解其逻辑非常重要，因为：

* 操作任务需要末端与目标对齐
* 可视化调试需要查看末端轨迹
* 校准流程需要使用末端位姿

---

## 0.3 逆向运动学（IK）

逆向运动学的目的：

> 给定末端执行器的目标位置与方向，求出机器人各关节角度。

仓库中包含：

* 数值迭代的 IK 示例
* 自己写一个简化 IK 求解器
* 如何在仿真中调用 Isaac Lab 的 IK API

IK 在以下任务中非常重要：

* 抓取姿态求解
* 抽屉开关、开门等目标姿态约束
* 强化学习训练前的初始控制策略

---

## 0.4 雅可比矩阵（Jacobian）

Jacobian 描述：

> 关节速度如何影响末端的线速度和角速度。

你将学习：

* 如何理解关节运动对末端速度的影响
* 为什么在力控、阻抗控制中必须用 Jacobian
* 如何用 Jacobian 避免奇异点

Notebook 提供实际可运行的可视化：移动关节、查看末端速度的变化。

---

## 0.5 URDF 文件（机器人模型结构）

URDF（统一机器人描述格式）用于描述：

* 机器人的关节（类型、自由度、限制）
* 连杆（外形、质量、惯性）
* 可视化和碰撞几何体
* 传感器和工具链接点

仓库中包含：

* 简单两连杆机械臂 URDF 示例
* Franka URDF 示例
* 如何将 URDF 转为 USD 供 Isaac Lab 使用

你将学习：

* 关节类型（旋转、平移、固定）
* 机械臂父子结构如何定义
* 机械臂每个 link 的坐标系意义

这部分内容对自建仿真机器人非常关键。
